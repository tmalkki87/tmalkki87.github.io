TCP ja UDP

  • TCP = Transmission Control Protocol
  • UDP = User Datagram Protocol
  • sijaitsevat kuljetuskerroksella sekä OSI- että TCP/IP-mallissa
  • monet ylemmän kerroksen protokollat rakentuvat näiden päälle

TCP

  • koneet käyvät aktiivista keskustelua yhteyden aikana
  • pakettien perilletulo ja oikea järjestys varmistetaan
  • --> luotettavuus; hyvä tapa välittää tärkeitä tietoja
  • myös toimintoja ruuhkanhallintaan

TCP

Käytetään mm:

  • HTTP
  • FTP
  • SMTP
  • SSH

UDP

  • yhteydetön viestintätapa; vastaanottajan saatavuutta ei etukäteen tarkisteta
  • pakettien perilletuloa ei myöskään varmisteta
  • UDP-paketista puuttuu TCP-paketin ylimääräiset varmistusosat, mikä tekee siitä kevyemmän
  • paketit voidaan yleislähettää

UDP

Käyttökohteita:

  • DNS-pyynnöt
  • median suoratoisto
  • verkkopelit
